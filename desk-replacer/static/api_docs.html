Obter token de Autenticação

Use este endpoint para gerar um token JWT temporário para chamadas autenticadas. O token expira em 24 horas.

POST https://api.vagas.com/v1/auth/token

key Headers 

Content-Type: application/json
X-API-KEY: sua_chave_de_acesso

Parâmetros de Requisição (JSON)

{
  "client_id": "ID_DO_CLIENTE",
  "client_secret": "SEGREDO_DO_CLIENTE",
  "grant_type": "client_credentials"
}

check_circle Resposta de Sucesso 

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR...",
  "token_type": "Bearer",
  "expires_in": 86400
}
